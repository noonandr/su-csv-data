# Suade CSV data
* SPWA that displays a table and two graphs when CSV data is uploaded*

## Installation

cd into root folder:

```
$ cd su-csv-data
```

Install npm modules:


```
$ npm install
```

## Run Server

To run the server:

```
$ npm run serve
```

Test data:

```
$ vi testdata.csv
```
